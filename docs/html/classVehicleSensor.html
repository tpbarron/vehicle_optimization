Detailed Description

Emits data from vehicle

I'll going to make some assumptions about what this sensor knows for simplicity.

Speed Direction Exact position Road (in real application would likely need to be calculated from position) Lane on road Current acceleration Current steering wheel movement Degree of brake currently applied
